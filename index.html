5 Serveurs MCP ✓ fait

Branche les super-pouvoirs connectés de Claude Code.

claude mcp add --transport http vercel https://mcp.vercel.com

Pour les serveurs OAuth (Vercel, Stripe) : ajoute-les, puis dans Claude tape /mcp → Authenticate. Tu peux aussi copier templates/mcp.json.example en .mcp.json.

7 Fichiers Claude & centralisation ✓ fait

Range chaque chose au bon niveau : global (toi partout) vs projet (ce repo) vs local (perso, non commit).

🌍 Global — ~/.claude/CLAUDE.md (ton style), settings.json, commands/, agents/. MCP user dans ~/.claude.json
📦 Projet — racine + .claude/CLAUDE.md (stack, règles), .mcp.json, .claude/settings.json (commit, partagé équipe)
🔒 Local — .claude/settings.local.jsonTes overrides perso (gitignored)
♻️ Anti-duplicationUne règle commune ? → en global. Importe avec @chemin au lieu de copier-coller
Règle d'or : écris chaque règle une seule fois, au bon niveau. Jamais de secret dans un fichier commit (CLAUDE.md, .mcp.json) → toujours ${VAR} + .env.local.
Compris la hiérarchie global / projet / local + l'ordre de priorité
